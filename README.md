# Face-Recognition-based-Attendance-System


## Features :clipboard:
---------------------------
* Check Camera
* Capture Faces
* Train Faces
* Recognize Faces & Attendance
* Automatic Email


## Tech Stack Used :computer:
--------------------------
**Build With -** 
* Python 3.8

**Modules Used -**

* OpenCV Contrib
* Pillow
* Numpy
* Pandas
* Shutil
* CSV
* yagmail
* Tkinter

**Facial Recognition Algorithms -**
* Haar Cascade
* LBPH (Local Binary Pattern Histogram)

**Softwares Used -**
* Pycharm 2019.2
* VS CODE 
* Jupyter Notebook
* Git
